The Datasaurus Dozen

Description:

This data set, "The Datasaurus Dozen (Long)", provides a set of data sets that

"retain the same summary statistics despite having radically different distributions. The data set represents a larger and quirkier version of the object lesson than is typically taught via Anscombe's Quartet." (Locke & McGowan, 2018)

The data set contains twelve variations of two variables (x and y). This version of the data set is presented in a "long form", and as such contains three columns.

Variables:

  • dataset - name of the variation of the data set.
  • x - values of variable x
  • y - values of variable y

References:

Locke, S. & McGowan, L. D. A. (2018). datasauRus: Datasets from the Datasaurus Dozen. R package version 0.1.4. https://CRAN.R-project.org/package=datasauRus

Matejka, J., & Fitzmaurice, G. (2017, May). Same stats, different graphs: Generating datasets with varied appearance and identical statistics through simulated annealing. In Proceedings of the 2017 CHI Conference on Human Factors in Computing Systems (pp. 1290-1294).
