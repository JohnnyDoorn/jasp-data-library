Dancing Cats: Chi-squared Test of Independence

Description:

This fictional data set, "Dancing Cats", provides the number of cats who learned to dance, trained by one of two types of conditioning.

Variables:

  • Training - Type of conditioning (Affection as Reward, Food as Reward).
  • Dance - Did the cat learn to dance? (Yes, No).
  • Counts - The number of cats belonging to the category.

This example JASP file demonstrates the use of Chi-squared test of independence. Specifically, we assess the adequacy of the null hypothesis that both types of conditioning are equally efficient in training cats to dance.

Reference:

Field, A. P. (2017). Discovering Statistics Using IBM SPSS Statistics (5th ed.). London: Sage.
