Neighborhood-Based Clustering

This example shows a k-means cluster analysis for identifying differences among Iris flower types based on the length and the width of their sepals and petals. The data set contains 150 observations of three iris flower types: Setosa, Versicolor, and Virginica. Four features were measured from each flower in the data: the length and the width of the sepals and petals, in centimeters. 

  • Sepal.Length: The length of the flower's sepal in cm.
  • Sepal.Width: The width of the flower's sepal in cm.
  • Petal.Length: The length of the flower's petal in cm.
  • Petal.Width: The width of the flower's petalin cm.

Because we know that there are 3 different flower types in the data, we build a fixed 3-cluster model on the data set.
