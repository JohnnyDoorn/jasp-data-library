Distributions: Normal distribution

Description:

This fictional data set, "Distributions", provides variables generated from eight continous and four discrete probability distributions.

Variable names indicate the probability distribution it was simulated from, the number of samples (10, 100, or 1000), and the parameter values in parentheses.

The purpose of this JASP file is do demonstrate the use of the Discover Distributions module. Specifically, we will focus on one particular distribution: the Normal (Gaussian) distribution. First, we will investigate its theoretical defininition. Then, we try to fit the Normal distribution to data. Then, we assess whether the Normal distribution fits the data.
