Support Vector Machine Classification

This example shows a support vector machines classification analysis for identifying labels in a two-dimensional spiral figure based on its coordinates. The data set contains 200 observations of the target variable:

  • Color: Red and Black.

Two features were available from each point on the plane:

  • x: The x-coordinate of the point
  • y: The y-coordinate of the point

The Support Vector Machine model table shows some of the used parameters for the algorithm, along with the sample sizes for the different subsets of the data. Since the algorithm is not optimized, only the test set accuracy is shown.

Data Split

As can be seen, 160 observations are used for training the algorithm. The remaining 40 observations are used as holdout test data to assess the performance on the algorithm.

The confusion matrix gives insight into the prediction error on the test set, and shows how the included observations were predicted by the model. It shows that for 13 + 19 = 32 of these observations, the observed class matched the predicted class, resulting in a test accuracy of 32 / 40 = 0.8.
