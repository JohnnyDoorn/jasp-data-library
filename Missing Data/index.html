Missing Data: How JASP Deals with Missing Values

Description:

This data set, "Missing Data", provides synthetic data with missing values. Example created by Alexander Ly.

This example showcases how JASP deals with missing values.

See a post on the JASP forum: http://forum.cogsci.nl/index.php?p=/discussion/3568/how-does-jasp-handle-missing-values-exactly

Bayesian Paired Samples T-Test

Exclude pairwise:

In a paired samples t-test, the pairwise option removes only the missing observations of variables that are directly involved in the individual comparison.

The test varA - varB is done without the first two cases, because varA is missing the first case and varB is missing the second case. Thus, the results are calculated based on 9-2=7 data points. The same applies for comparison varB - varC (excluding cases 2 and 3), varC - varD (excluding cases 3 and 4), and varD - varE (excluding cases 4 and 5). All these comparisons are based on 7 data poins

Note however, that for the comparison varE - varF only the fifth case is removed. Thus, the results are calculated based on 9-1=8 data points.

Bayesian Paired Samples T-Test

varA - varB 4.740 0.006
varB - varC 0.367 0.003
varC - varD 0.548 0.009
varD - varE 0.683 0.013
varE - varF 0.690 0.015


Bayesian Paired Samples T-Test

Exclude listwise:

In a paired samples t-test, the listwise option removes all missing observations of variables that are involved in any of the specified comparisons. ** **

This results in equal number of observations for all comparisons (we remove the first five cases from all comparisons).

Measure 1   Measure 2 BF₁₀ error %
varA - varB 2.900 0.001
varB - varC 0.428 7.958×10-4
varC - varD 0.456 0.002
varD - varE 1.133 0.012


Bayesian Independent Samples T-Test

Exclude cases analysis by analysis:

The independent samples t-test with exclude analysis by analysis is equivalent to the pairwise option in a paired t-test. 

For the dependent variable varA, only the first case is excluded, for varB, only the second is excluded, for varC, only the third case is excluded.

Bayesian Independent Samples T-Test

Exclude listwise:

The independent samples t-test with listwise exclusion is equivalent to the listwise option in a paired t-test. ** **

The list here consists of varA, varB, varC. The first three cases are excluded from all three t-tests, as varA is missing value in the first case, varB in the second, and varC in the third.
