Process Capability Study

Initial capability study evaluation


Basic information about the study


This initial process capability study[1] has been performed in Luton at Channel 2 for the bore grinding process. The machine that produced the bearings is machine SHD 110F - No. 3421. The parameter under study is the inner ring diameter (dm). The tolerance for this parameter (dm) is -16/0 µm. The target of the process is -8 µm. The sample size (N) is 100 parts. The rational subgroup size (n) is five parts. The subgroups for the initial capability study are collected during two shifts (28 July), once per hour.


Data verification


A quick glance at the histogram and probability plot, both covering the tolerance range, will normally reveal any anomalous data. If found, it may be necessary to re-measure some individual parts, or the whole sample. In other cases it may be just enough to correct obvious recording errors.


[1]   Data is taken from the SKF Quality Techniques, “QT 1 - Process capability studies” (PUB GQ/P9 10347/1 EN – January 2022), the analysis is performed by René Klerx

Control Chart

Control chart

  • Enter the data for each subgroup, such as shift, time, date, and individual measurements.
  • Create an X-bar & Range control chart, using the Phase 1 application of the control chart, i.e. the limits are based on the current data.
  • Calculate upper and lower control limits for averages (X-bar) and Ranges (R) using the formulae based on R‑bar. Choose the method for estimating standard deviation based on R-bar which estimates the standard deviation based on the average of the subgroup ranges.
  • Plot the control limits, by using red dashed lines. The central line should be a green solid line.


Stability

  • Interpret the Phase 1 control chart for statistical control, based on all six out-of-control signals.
  • If the completed control chart does not show any of the out-of-control signals, it can be said that the process is in a state of statistical control. Meeting any of these criteria means that your process is not in a state of statistical control, namely there are special causes of variation present. Corrective actions must now be implemented, and the initial capability study can then be repeated.

Histogram

Histogram or normal probability plot completion

  • Evaluate the shape of the distribution by analysing the histogram and the probability plot using the common guidelines. If the distribution is as assumed[2] (normal or skewed) the capability and performance indices may then be calculated.
  • Construct a histogram with the best fit probability density function of the normal distribution. Add on the graph specifications and target, by using dark purple solid thick lines for LSL and USL, and a green dashed line for the target.
  • Note good or poor fit. Look for skewness, truncation, bi-modality, genuine outliers, etc.
  • Create a normal probability plot to check normality.
  • If the histogram indicates a bell shaped curve but the Anderson-Darling test rejects normality[3], perform a Ryan-Joiner test for to check normality. In some special cases it is better to perform a Kolmogorov-Smirnov test to support the final decision: Normality or non-normality.
  • If the data follow the distribution as assumed (normal or skewed) indices may then be calculated.


[2]   The only reference for a particular shape of the distribution is previous data from the same process. If that is not available, further analysis is needed.

[3]   The Anderson-Darling test rejects normality if the resulting p-value is < 0.10.

The Anderson-Darling good-of-fit test is a test whether a given data set is following a certain probability distribution. The Anderson-Darling hypothesis is defined as:


H0:         The data follow a specified distribution.

HA:        The data do not follow the specified distribution.
